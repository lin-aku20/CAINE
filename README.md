## вңЁ CaracterГӯsticas

### рҹ§  Cerebro en la Nube вҖ” Sin Costo
Conectado a travГ©s de una API compatible con OpenAI. Funciona con **Google Gemini**, **OpenRouter** (modelos gratuitos), o cualquier proveedor de tu elecciГіn. Zero hardware local requerido para el LLM.

### рҹҺҷпёҸ Voz Bidireccional
- **STT** (Speech-to-Text): Vosk (offline) o Google STT (online).
- **TTS** (Text-to-Speech): Piper TTS con voces en espaГұol de alta calidad (Pablo/Davefx), SAPI5 o pyttsx3.
- **Wake Word** global: Di *"Caine"* o *"Despierta"* desde cualquier lugar y CAINE responde.

### рҹ‘ҒпёҸ Consciencia del Escritorio
CAINE captura capturas de pantalla periГіdicas, aplica OCR y anГЎlisis visual para entender quГ© estГЎs haciendo. Puede comentar, alertar o ayudar de forma proactiva basГЎndose en lo que ve.

### рҹ–ҘпёҸ Control Total del Sistema Operativo
| Comando | AcciГіn |
|---------|--------|
| `"Abre Chrome"` | Lanza aplicaciones por nombre |
| `"Abre la carpeta Escritorio"` | Navega carpetas del sistema |
| `"Apaga la PC"` | Apagado del equipo |
| `"Corta la llamada"` | Cierra Discord, Zoom, Teams |
| `"Teclas Ctrl+Shift+Esc"` | EnvГӯa atajos de teclado |
| `"Busca en internet..."` | Abre bГәsquedas web |

### рҹ’ҫ Memoria Persistente
- **Memoria a corto plazo**: Historial de conversaciГіn por sesiГіn.
- **Memoria a largo plazo**: Base de datos SQLite con uso de aplicaciones, sesiones de juego y contexto acumulado.
- **Contexto del mundo real**: Registra hГЎbitos, apps favoritas y tiempo de uso para personalizar el comportamiento.

### рҹҺЁ Interfaz Visual (Digital Circus)
Una UI de escritorio estilizada en modo oscuro con:
- Avatar animado que reacciona al estado (escuchando, pensando, hablando, dormido)
- Chat en tiempo real con CAINE
- Controles: **HABLAR**, **ENVIAR**, **MUTE**, **DORMIR**, **VISTAZO**
- Borde neГіn que pulsa segГәn el estado actual

### вҡҷпёҸ AutonomГӯa Configurable
CAINE puede comentar proactivamente sobre lo que ve en pantalla, detectar cuando estГЎs en un juego, avisar sobre inactividad prolongada o generar pensamientos propios cuando el sistema estГЎ quieto.

## рҹҡҖ InstalaciГіn RГЎpida

### Requisitos
- Windows 10/11
- Python 3.11+
- MicrГіfono (opcional pero recomendado)
- [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki) (opcional, para anГЎlisis de pantalla)

### Setup

```powershell

git clone https://github.com/linmelin/CAINE.git
cd CAINE

# 2. Ejecutar el setup automГЎtico
.\setup_caine_environment.ps1

# 3. Configurar tu proveedor de IA en config/config.yaml
```

### ConfiguraciГіn del Cerebro (`config/config.yaml`)

**OpciГіn A вҖ” Google Gemini (gratis, recomendado):**
```yaml

  base_url: https://generativelanguage.googleapis.com/v1beta/openai/
  primary_model: gemini-2.5-flash
  api_key: TU_GOOGLE_AI_STUDIO_KEY
```

**OpciГіn B вҖ” OpenClaw (local, con cualquier proveedor):**
```yaml
ollama:
  base_url: http://127.0.0.1:18789/v1
  primary_model: openclaw
  api_key: ""
```

**OpciГіn C вҖ” OpenRouter (multi-modelo):**
```yaml
ollama:
  base_url: https://openrouter.ai/api/v1
  primary_model: meta-llama/llama-3.3-70b-instruct:free
  api_key: TU_OPENROUTER_KEY
```

### Lanzar CAINE

```powershell
.\Start_CAINE.bat
```

## рҹҺ® CГіmo Usar

## вҡҷпёҸ ConfiguraciГіn Avanzada

### Estados de CAINE
```
SLEEP           вҶ’ En standby, escuchando wake word
WAITING_FOR_USER вҶ’ TerminГі de hablar, espera tu respuesta
LISTENING       вҶ’ Capturando tu voz
THINKING        вҶ’ Procesando con el LLM
SPEAKING        вҶ’ Reproduciendo audio
OBSERVING       вҶ’ Analizando la pantalla
EXCITED         вҶ’ Evento importante detectado (ej: juego)
ACTING          вҶ’ Ejecutando acciГіn en el sistema
```

### Variables Clave en `config.yaml`
```yaml
desktop:
  always_listen_microphone: true   # Modo continuo (sin wake word)
  use_piper_voice: true            # Voz de alta calidad
  presence_interval_seconds: 75   # Frecuencia de comentarios autГіnomos

autonomy:
  max_commentary_per_hour: 10      # LГӯmite de intervenciones
  commentary_cooldown_seconds: 90  # Tiempo entre comentarios

voice:
  wake_variants:
    - caine
    - despierta
    - hey caine
```

### Instalar como Servicio de Windows (24/7)
```powershell
# Ejecutar como Administrador
.\scripts\install_service.ps1
```
